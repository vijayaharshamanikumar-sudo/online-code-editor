# Online Code Editor

## Project Description

The Online Code Editor is a simple web application that allows users to write HTML, CSS, and JavaScript code in separate editors and view the output instantly in a Live Preview section. This project is designed for beginners to practice web development in an easy and interactive way.

## Features

- HTML Editor
- CSS Editor
- JavaScript Editor
- Live Preview
- Run Button
- Clear Button
- Simple and User-Friendly Interface

## Technologies Used

- HTML
- CSS
- JavaScript

```
Online-Code-Editor/

```

## How to Run the Project

1. Download or copy the project files.
2. Open the project folder.
3. Double-click **index.html** or open it in any web browser.
4. The Online Code Editor will open.
5. Enter your code in the editor panels.
6. Click the **Run** button to view the output.
7. Click the **Clear** button to remove all code and start again.

## Working

1. Open the website.
2. Enter code in the editors.
3. Click **Run**.
4. The application displays the output in the Live Preview section.
5. Edit the code if required and click **Run** again.
6. Use **Clear** to reset the editors.
